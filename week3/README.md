# week3 - 樂呵呵健身房-購買課程

### 安裝相依套件

請先確認已安裝 Node.js 與 TypeScript，然後執行下列命令來安裝必要套件（若有相關 package.json 時）：

````bash
npm install
````

### 執行系統
```bash
npm run dev
````
## 系統指令

- **使用預設資料**: `--default`
- **清空資料指令**: `--clear`
- **結束指令**: `--exit`
- **獲取總金額**: `--cal`
- **篩選為購買**: `--filter`
- **顯示目前歷史資料**: `--his`
- **顯示指令**: `--help`
